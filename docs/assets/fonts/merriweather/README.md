Merriweather Regular is vendored from the repository-level Typst font bundle.

Source family:
- `fonts/merriweather/Merriweather-Regular.ttf`
- `fonts/merriweather/OFL.txt`

This local webfont is used by the archive page date labels so the browser can
render Merriweather consistently on GitHub Pages without relying on system font
availability.
