Merriweather font files vendored from the official upstream repository:
https://github.com/SorkinType/Merriweather

Included files:
- Merriweather-Regular.ttf
- Merriweather-Italic.ttf
- Merriweather-Bold.ttf
- Merriweather-BoldItalic.ttf
- Merriweather-Light.ttf
- Merriweather-LightItalic.ttf
- Merriweather-Black.ttf
- Merriweather-BlackItalic.ttf
- OFL.txt

These fonts are loaded by `scripts/publish_posts.py` via Typst's `--font-path`
option so GitHub Actions and local builds can use the same English serif font.
